# Question: Find GP

Create a class to calculate the terms of a Geometric Progression (GP).

The program should accept three integer inputs: - `a` -- the first term
of the GP - `b` -- the number of terms - `c` -- a value used in
generating the progression

If the number of terms (`b`) is less than or equal to `0`, print:

`It is not possible`

Otherwise, generate and display the GP sequence.
